Workflow 1: Quick Study Mode
Use this when user says: I want to study balance sheet quickly
Workflow :
1. user clicks on quick study 
2. user enters a topic
3. learning path agent
4. Vector DB Tool retrieves materials
5. Learning Path Tool creates main points
6. Adaptive Quiz Agent
7. Quiz Tool generates small quiz
8. Export Tool creates summary + quiz PDF
So Quick Study does not need separate agent. it uses tools from other agents that are already present also uses primarly Learning Path Agent + Adaptive Quiz Agent


Workflow 2: Full Study Mode (Use this when user wants complete learning plan.)
workflow:
1. User enters topic.
2. Knowledge Assessment Agent
3. Profile DB Tool gets/creates learner profile (profile extraction : customer customization)
4. Vector DB Tool retrieves diagnostic questions.
5. Quiz Tool creates diagnostic quiz
6. Quiz Tool grades quiz
7. Profile DB Tool stores score.
8. Learning Path Agent.
9. Learning Path Tool finds gaps
10. Learning Path Tool creates ordered path
11. Adaptive Quiz Agent
12. Quiz Tool generates adaptive quiz.
13. User answers quiz
14. Quiz Tool grades quiz.
15. Mastery Tracking Agent
16. Mastery Tool updates mastery
17. If mastery < 80%, repeat quiz/review (provide answers needs more)
18. If mastery >= 80%, Export Tool creates report

Simple Agent Responsibility
Knowledge Assessment Agent
- starts learner profile
- gives diagnostic quiz
- stores first performance

Learning Path Agent
- retrieves material
- finds gaps
- creates learning path
- creates quick study notes

Adaptive Quiz Agent
- generates quiz
- changes difficulty
- grades answers

Mastery Tracking Agent
- updates mastery score
- decides repeat or complete
- triggers final report
